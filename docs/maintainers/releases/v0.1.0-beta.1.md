# localwp-mcp v0.1.0-beta.1

`localwp-mcp` is a LocalWP-aware MCP server for WordPress developers.

This first beta release focuses on the core developer workflow: discover Local sites, inspect site health, run WP-CLI against the correct Local runtime, query or modify the local database, and manage backups and restore flows through one MCP server.

## Highlights

- Two simple access profiles:
  - `safe`
  - `full-access`
- Local site discovery from Local's own metadata
- Site-aware WP-CLI execution
- Safe SQL reads and full SQL access when explicitly enabled
- Database export and import
- Local-friendly backups and restore flows
- Local logs and doctor-style diagnostics
- MCP resources and prompts for better agent workflows

## Platform Status

- macOS: live-tested against a real LocalWP site
- Windows: platform-aware support is implemented; live validation is planned
- Linux: platform-aware support is implemented through path logic and tests

## Notes

- `safe` is the default profile
- `full-access` is intended for local development when you want the agent to fully work on the site and database
- full backups are folder-based for more predictable behavior across platforms

## Known Limitations

- Windows has not yet been live-tested in the same depth as macOS
- Linux support is code-validated rather than live-tested on a LocalWP install
